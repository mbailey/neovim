# LazyVim

## Top level commands

LazyVim provides several commands for managing plugins and configuration. These commands can be accessed through the LazyVim UI by pressing `:Lazy`.

- **Home (H)**: Navigate to the home/dashboard screen of LazyVim
- **Install (I)**: Install or update plugins
- **Update (U)**: Check for and apply updates to plugins
- **Sync (S)**: Synchronize plugins (install, clean, and update)
- **Clean (X)**: Remove unused or disabled plugins
- **Check (C)**: Check plugin status and health
- **Log (L)**: View plugin installation and operation logs
- **Restore (R)**: Restore plugins to a previous state
- **Profile (P)**: View performance profiling information
- **Debug (D)**: Access debugging tools and information
- **Help (?)**: Display help and documentation

## Common keybindings

LazyVim comes with many useful keybindings out of the box:

### General

- `<Space>` - Leader key
- `<Space>h` - Open dashboard
- `<Space>ff` - Find files
- `<Space>fg` - Live grep (search in files)
- `<Space>fb` - Browse buffers

### Navigation

- `<Ctrl-o>` - Jump back
- `<Ctrl-i>` - Jump forward
- `gd` - Go to definition
- `gr` - Go to references

### Editing

- `<Space>cc` - Comment line
- `<Space>gc` - Comment selection
- `<Space>w` - Save file
- `<Space>q` - Quit

### Windows and Buffers

- `<Ctrl-h/j/k/l>` - Navigate between windows
- `<Space>bd` - Delete buffer
- `<Space>bn` - Next buffer
- `<Space>bp` - Previous buffer
